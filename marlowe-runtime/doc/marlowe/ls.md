# List the Contracts Being Tracked

```console
Usage: marlowe ls [-s|--show-status] [-f|--show-failed] [-a|--all]

  List managed contracts

Available options:
  -s,--show-status         Show the status of the contract as well as its ID
  -f,--show-failed         Include contracts whose follower encountered an error
  -a,--all                 Show all Marlowe contracts on chain.
  -h,--help                Show this help text
```
